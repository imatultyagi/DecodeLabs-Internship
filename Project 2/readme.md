# E-Commerce Sales Data Analysis Using Excel

## Overview

This project analyzes an e-commerce sales dataset using Microsoft Excel to identify patterns, trends, and distributions in transaction data.

The analysis was performed using descriptive statistics, Pivot Tables, charts, and observation-based reporting to derive meaningful insights from 1200 transaction records.

---

## Objective

* Analyze transaction data to understand patterns and trends.
* Calculate descriptive statistics such as mean, median, minimum, and maximum.
* Examine customer behavior through payment methods and referral sources.
* Analyze order status distribution.
* Summarize findings through observations and visualizations.

---

## Dataset Information

The dataset contains 1200 e-commerce transaction records with the following attributes:

* Order ID
* Date
* Customer ID
* Product
* Quantity
* Unit Price
* Payment Method
* Order Status
* Coupon Code
* Referral Source
* Total Price

---

## Analysis Performed

### 1. Basic Statistics

Calculated descriptive statistics for:

* Quantity
* Unit Price
* Items In Cart
* Total Price

Metrics analyzed:

* Count
* Average (Mean)
* Median
* Minimum
* Maximum

### 2. Payment Method Distribution

Analyzed transaction counts across different payment methods to understand customer payment preferences.

### 3. Order Status Distribution

Examined the frequency of different order statuses to understand order processing patterns.

### 4. Referral Source Distribution

Analyzed customer acquisition channels and compared referral source contributions.

### 5. Key Findings

Summarized important observations derived from statistical and distribution analysis.

---

## Key Observations

* A total of 1200 transaction records were analyzed.
* The average quantity purchased per order was 2.95 units.
* The average order value was 1053.97.
* Online payment was the most frequently used payment method.
* Cancelled orders were the most common order status.
* Instagram generated the highest number of referrals.
* Customer activity was distributed across multiple payment and referral channels.
